# InspectIC

Inspection dashboard for Converge's quality lab. Search, filter, sort, and review electronic component inspection records.

## Stack

- **Vite** + **React 19** + **TypeScript**
- **Tailwind CSS v4** with custom brand theme
- **Lucide React** for icons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/` — ready for Vercel deployment.

## Data

Sample inspection records are hardcoded in `src/data/inspectionData.json` (4 records). No backend required.
